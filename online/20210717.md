#### 基本环境

-. 管理网：172.16.34.240/16
-. 存储前段: 192.168.34.0/24
-. 存储后端：100.100.0.0/16
- 13个mds，6主7备。


1. 官网下载iperf的安装包
```
https://iperf.fr/iperf-download.php
```
2. 安装之后，在集群节点上用`iperf` 以 `-s`模式运行
```
// start the iperf server
iperf -s
```

3. 在客户端上，iperf3 以客户端运行，测带宽不需要参数：
```
iperf3 -c 182.200.21.74
```
